# Project CRUD sederhana laravel & vue 

## Instalasi
Clone repository:

```
git clone https://github.com/deniace/Laravel_vue_crud.git
cd crud_vue
```

buat konfigurasi `.env`

nama database crud_vue

## Install Package

```
composer install
npm install
npm run dev
```

## jalankan migrasi
```
php artisan migrate --seed
```

## Jalankan server 
```
php artisan serve
```
